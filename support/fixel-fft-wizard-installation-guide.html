Fixel FFT Wizard - Installation Guide

How to Install Fixel FFT Wizard

Fixel FFT Wizard Product and File Structure

What Is Fixel FFT Wizard?
Fixel FFT Wizard is a free, simple and effective Pattern Suppression FFT Plug In for Photoshop.
Product Download File Structure
When a user downloads the product it downloads a ZIP file which contain the following structure:
  • Fixel FFT Wizard <MajorVerion>.<MinorVersion>.<BuildNumber>.zip - The ZIP File container.
    • Fixel FFT Wizard <MajorVerion>.8bf - Windows Photoshop Plug In.
    • Fixel FFT Wizard <MajorVerion>.plugin - macOS Photoshop Plug In.
    • Fixel FFT Wizard - Installation Guide.url - Link to Fixel FFT Wizard Installation Guide.
    • Fixel FFT Wizard - Introductory Blog Post.url - Link to Fixel FFT Wizard Introductory Blog Post.
    • Fixel FFT Wizard - Product Page.url - Link to Fixel FFT Wizard Product Page.
    • Fixel FFT Wizard Action.atn - Fixel FFT Wizard Automated Pattern Suppression Photoshop Action (Curtsy of Jonas Madsen Rogne).
    • Fixel FFT Wizard Backward.jsx - Fixel FFT Wizard Auxiliary Script - Backward Step - Photoshop Extend Script.
    • Fixel FFT Wizard Create Luminsoity Image.jsx - Fixel FFT Wizard Auxiliary Script - Create Luminosity Image - Photoshop Extend Script.
    • Fixel FFT Wizard Forward.jsx - Fixel FFT Wizard Auxiliary Script - Forward Step - Photoshop Extend Script.
    • Fixel FFT Wizard UI <MajorVerion>.jsx - Fixel FFT Wizard UI - Photoshop ExtendScript.

Fixel FFT Wizard - Installation

Installation Procedure
The installation has 3 mains steps: Installing the Plug In, installing the UI and Auxiliary Extend Scripts and installing the Action.
All following steps assume the use has closed Photoshop and extracted the ZIP file so each file is available.

Installing the UI and Auxiliary Scripts:
  1. Copy the HTML5 UI file Fixel FFT Wizard UI <MajorVerion>.jsx (For instance, for Fixel FFT Wizard 1 it will be Fixel FFT Wizard UI 1.jsx) to:
    • Windows - C:\Program Files\Adobe\Adobe Photoshop \Presets\Scripts\Fixel Algorithms\.
    • macOS - /Application/Adobe Photoshop /Presets/Scripts/Fixel Algorithms/.
  2. Copy the Auxiliary Scripts files: Fixel FFT Wizard Backward.jsx, Fixel FFT Wizard Create Luminsoity Image.jsx and Fixel FFT Wizard Forward.jsx to (Same folder as the UI):
    • Windows - C:\Program Files\Adobe\Adobe Photoshop \Presets\Scripts\Fixel Algorithms\.
    • macOS - /Application/Adobe Photoshop /Presets/Scripts/Fixel Algorithms/.
    Create the folder Fixel Algorithms if it does not exist. Copying might require Administrator Privileges. If you upgrade the version you might need to authorize overwrite on the existing files.

Installing the Photoshop Plug In:
  1. Copy the Plug In File (Windows - Fixel FFT Wizrd <MajorVersion>.8bf, macOS - Fixel FFT Wizard <MajorVersion>.plugin) to:
    • Windows - C:\Program Files\Common Files\Adobe\Plug-ins\CC\Fixel Algorithms\.
    • macOS - /Library/Application Support/Adobe/Plug-ins/CC/Fixel Algorithms/.
    Create the folder Fixel Algorithms if it does not exist. Copying might require Administrator Privileges. If you upgrade the version you might need to authorize overwrite on the existing files.
Installing the Automated Pattern Suppression Photoshop Action:
  1. Launch Photoshop.
  2. Open the Window Menu in Photoshop Choose Actions from the menu.
  3. Click the Menu Icon.
  4. Scroll Down to Load Actions.
  5. Select Fixel FFT Wizard Action.atn.
    6. You may have a look at PHLEARN - How to Install Photoshop Actions for visual guide.
    You may also have a look at About Actions and the Actions Panel for more information.


Once done installing both, launch Adobe Photoshop and start using Fixel Recolor.

Fixel Fixel FFT Wizard - Installation & First Use Frequently Asked Questions (FAQ)

How to Launch the Fixel Recolor UI?
Launching the UI is done like launching any HTML Panel in Photoshop: Window -> Extensions -> Fixel Recolor.


If you can't find it there, it means the UI wasn't installed properly.
What Is Fixel Algorithms -> Fixel Color Analyzer x.x.xxx Under the Filter Menu?
As written above, Fixel Recolor use innovative algorithm to analyze an image and extracts its color palette. The filter Fixel Color Analyzer in the Filter Menu is used by Fixel Recolor to apply the algorithm on the user image. Since no such analysis is available in Photoshop we had to develop and implement the algorithm by ourselves. It is good it is there as it means the Plug In installation is successful. Yet the user shouldn't use it directly (Only use it through th UI).
The macOS Installer Won't Run
After double clicking the dmg file in the open window use Right Click + Open to launch the installer pkg installer.
